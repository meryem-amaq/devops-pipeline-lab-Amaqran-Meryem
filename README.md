# Automatisation IA

Landing page front-end pour une offre d'automatisation IA, inspiree d'un hero sombre et lumineux de type produit SaaS.

## Structure

- `index.html` : page principale
- `src/styles.css` : style visuel et responsive
- `src/main.js` : interactions simples et animations d'apparition
- `tests/manual-checklist.md` : verifications manuelles front-end

## Utilisation

Ouvrez `index.html` dans un navigateur pour voir la maquette.

## Contenu

La page contient :

- un hero principal tres visuel
- une section solution
- une section workflow
- une section benefices
- un bloc contact / call-to-action
